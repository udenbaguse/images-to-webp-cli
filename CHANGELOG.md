# Changelog

All notable changes to this project will be documented in this file.

## [1.0.1] - 2026-04-17
f
- Fixed a bug where the CLI would crash when processing unsupported file formats

### Added
- Initial release of `images-to-webp-cli` CLI tool
- Convert single images to WebP format
- Convert all images in a directory
- Custom quality option (1-100, default: 80)
- Colored terminal output (success/error/info)
- Support for PNG, JPG, and JPEG input formats
- ES Modules configuration
